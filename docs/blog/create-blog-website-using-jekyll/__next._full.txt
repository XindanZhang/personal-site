12:T736,<p>Simply creating a blog website can be straightforward if you start from a theme built by someone else and customize it to fit your own writing.</p>
<h2 id="step-1-software-setup">Step 1: Software setup</h2>
<p>Install Ruby with Homebrew:</p>
<pre><code class="language-sh">brew install ruby
</code></pre>
<p>The prerequisite tools are <code>gcc</code> and <code>make</code>:</p>
<pre><code class="language-sh">brew install gcc make
</code></pre>
<p>Then add Ruby to your shell path:</p>
<pre><code class="language-sh">export PATH="/opt/homebrew/opt/ruby/bin:$PATH"
</code></pre>
<p>Reload your shell configuration:</p>
<pre><code class="language-sh">source ~/.zshrc
</code></pre>
<p>Install Bundler:</p>
<pre><code class="language-sh">gem install bundler
</code></pre>
<p>And make sure the Jekyll executable directory is available on your <code>PATH</code>:</p>
<pre><code class="language-sh">export PATH="$PATH:/Users/winifred/.gem/ruby/3.3.0/bin"
</code></pre>
<p>After any change to <code>Gemfile</code>, run:</p>
<pre><code class="language-sh">bundle install
</code></pre>
<h2 id="step-2-edit-the-theme-details">Step 2: Edit the theme details</h2>
<p>After cloning the repository from GitHub, start editing the Markdown files. Add a <code>_posts</code> folder at the root of the site, place blog posts inside it, and create a page like <code>blog.md</code> to collect them.</p>
<pre><code class="language-md">---
layout: page
permalink: /blog
permalink_name: /blog
title: "Your title"
---

# Your title

Add all the content here.
</code></pre>
<h2 id="step-3-upload-to-github-pages">Step 3: Upload to GitHub Pages</h2>
<p>Commit and push your changes:</p>
<pre><code class="language-sh">git add .
git commit -m "Describe your changes here"
git push origin main
</code></pre>
