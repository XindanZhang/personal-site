Add the search box to blog pages

2024-09-28#ui · #css · #blog

A tiny UI note on making a search field visible and usable for blog readers.

cd /blog
xindan@toronto-node:~$sed -n '1,$p' ./add-search-box-to-blog-pages.md

Add a search input to blog.md:

<input type="text" id="searchInput" placeholder="Search blogs..." />

Then style it to make it readable and easy to spot:

#searchInput {
  width: 100%;
  padding: 10px;
  margin: 20px 0;
  font-size: 16px;
  border: 2px solid #4a4a4a;
  border-radius: 4px;
  background-color: #f0f0f0;
  color: #333;
}

Simple details like spacing, contrast, and border treatment make a big difference for blog usability.
