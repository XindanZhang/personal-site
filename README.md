# Xindan Zhang — Personal Site

Single-page personal website styled after the molt.bot aesthetic.

## Local development

```sh
npm install
npm run dev
```

## Build for GitHub Pages

This project outputs to `docs/` for GitHub Pages on the `main` branch.

```sh
npm run build
```

Then commit and push. In GitHub Pages settings, select **Deploy from a branch** → `main` → `/docs`.
